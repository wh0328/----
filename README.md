## ⚡ 运行步骤

### 启动本地服务器
```bash
# 在项目文件夹中运行以下命令
python -m http.server 8000
# 或
python3 -m http.server 8000
```

### 访问网页
打开浏览器，访问以下地址：
```
http://localhost:8000
```

### 步骤3：体验功能
1. **浏览实验页面**：通过导航栏或实验入口按钮进入各个实验
2. **AI智能聊天**：在首页与AI助手对话
   - 使用真实的讯飞星火AI进行对话

## 🔧 配置说明


### 文件说明
- **index.html**：主页面，包含AI聊天功能
- **style.css**：页面样式文件
- **exp*.html**：各个实验的详细介绍页面
- **图片/**：存放页面使用的图片资源
- **视频/**：存放实验相关的视频文件
- **资料/**：存放实验资料和文档

## 🔍 故障排除

### 启动相关问题
1. **批处理文件无法运行**：
   - 右键"启动服务器.bat" → "以管理员身份运行"
   - 或使用 `python start_server.py`

2. **Python未找到**：
   - 确认已安装Python 3.x
   - 检查Python是否在系统PATH中

3. **端口被占用**：
   - 修改批处理文件中的端口号
   - 或使用命令：`python -m http.server 8080`

### 运行时问题
1. **权限错误**：确保有读取文件夹的权限
2. **浏览器缓存**：刷新页面或清除浏览器缓存
3. **页面显示异常**：确认所有文件都在同一目录

### 验证运行
启动服务器后，访问 `http://localhost:8000` 应该能看到：
- 多媒体实验平台首页
- AI聊天界面
- 导航菜单可以正常点击
