实验五：网页制作

网页制作目标：掌握HTML网页编辑技术，熟悉Dreamweaver软件，学会创建美观实用的多媒体网页。

快速跳转： 实验目的 | 实验内容及要求 | 参考资料 | 返回顶部

一、实验目的

  • 熟悉Dreamweaver CS6软件（可以选用最新版本）（也可以借助其他编辑器）
  • 掌握网页中文本格式的编辑
  • 掌握多媒体元素的插入及格式编辑
  • 掌握在网页中创建文本的超链接的方法
  • 掌握利用图像热点区域创建超链接的方法
  • 掌握锚记的命名以及锚链接的创建方法

参考学习W3school上的内容HTML和HTML5：
https://www.w3school.com.cn/html/html5_intro.asp

二、实验内容及要求

  • 制作"多媒体课程实验在线平台"，包含主网页（index.html）和子网页（注意命名方便识别），要美观大方
  • 主网页面设计风格不限，包含课程名及几次实验名称，其他元素可选择添加
  • 子网页，每个子网页为独立的实验介绍，如：实验目的，实验内容，实验方法及步骤等可以适当上传自己的作品作为实验的参考效果，注意文件不要太大
  • 每个子网页都可以返回主页及跳转到其他子网页

补充说明：（HTML版本不限）

  • 实验之前，可以创建站点（文件夹），包含图像文件夹，音频视频文件夹等
  • 网页页面的布局情况，可以适当借助表格或分割线等，如熟悉框架结构，DIV+CSS的都可以采用
  • 媒体的插入及设置等，可以参考相关代码
  • 最后递交设计好的网站等材料（不用写实验报告，只用递交网页源文件+实验体会）

三、参考资料与技术要点

HTML基础知识

  • 文本格式编辑：使用各种HTML标签控制文本样式、字体、大小、颜色等
  • 多媒体元素：插入图片、音频、视频等多媒体内容
  • 超链接：创建文本链接和图像热点区域链接
  • 锚链接：页面内部跳转和跨页面跳转

网页布局技术

  • 表格布局（table）
  • DIV+CSS布局
  • 框架结构（frameset）
  • 响应式设计

开发工具

  • Dreamweaver（推荐）
  • Visual Studio Code
  • Sublime Text
  • 其他HTML编辑器
